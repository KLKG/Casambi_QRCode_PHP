# Casambi_QRCode_PHP
QR Code Scanner in PHP

Benötigt unter Chrome eine SSL Verbindung für die Webcam.

--------------------------------------------------------------------------------------

Code Type:
0 = Empty
1 = Level
2 = Tc
3 = RGBW

Target Type:
0 = Broadcast
1 = Device
2 = Group
4 = Scene

Target ID:
Broadcast => 0
Device => 1-250
Group => 0 = ungrouped, 1-255 = Group
Scene => 1-255 = Group
